# UI Components

Reusable React components and design system for Junior Golfers Kenya applications.

## Components

- Form components (Input, Button, Select)
- Layout components (Header, Footer, Sidebar)
- Data display (Table, Card, Badge)
- Feedback (Alert, Modal, Toast)
- Navigation (Breadcrumb, Pagination)

## Development

```bash
# Start Storybook
npm run storybook

# Build components
npm run build
```

## Usage

```typescript
import { Button, Input, Card } from '@junior-golf/ui';
```
