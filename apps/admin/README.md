# Admin Dashboard

Next.js admin dashboard for managing Junior Golfers Kenya membership system.

## Features

- User management and verification
- Subscription management
- Event creation and management
- Analytics and reporting
- Club management

## Development

```bash
# Install dependencies
npm install

# Start development server (port 3001)
npm run dev

# Build for production
npm run build
```

## Access Control

Admin dashboard requires elevated permissions and is protected by role-based access control.
